## Comment
评论Model

#### 示例
```json
 
{
    "id": 2,
    "user_id": 3,
    "good_id": 1,
    "content": "测试评论",
    "evaluation": "Positive",
    "create_at": 1541582649,
    "update_at": 1541582649,
}
 
```

#### Property
| 名称 |类型|备注|
|---|---|---|---|
| id | number |  评论ID |
| user_id | number |  用户ID |
| good_id | number |  商品ID |
| content | string |  评价内容 |
| evaluation | string |  评价 |
| create_at | number |  创建时间 |
| update_at | number |  修改时间 |
