## 获取评论列表
创建评论 

### Request
url:`/api/good/:goodId/comments [get]`

#### Authorization
需要在Header添加身份信息

#### Parameters

| 名称 |类型|必须|备注|
|---|---|---|---|
| good | path | 是 | 商品ID |


#### ReqeustBody

##### Example
```json
{
	"content":"some comment with this good",
	"evaluation":"Positive"
}
```

##### Property

| 名称 |类型|备注|
|---|---|---|---|
| content | string |  评论内容 |
| evaluation | string |  评价 |

---

#### Example

{
    "id": 6,
    "user_id": 3,

    "content": "some comment with this good",
    "evaluation": "Positive",

        {
