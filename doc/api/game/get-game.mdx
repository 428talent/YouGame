## Summary
获取游戏信息  

### Path
`/api/game/:gameId [get]`

## Request


### Parameters

| 名称 |类型|必须|备注|
|---|---|---|---|
| :gameId | path | 是 | 要获取的游戏Id |

## Response

    "release_time": "2018-10-21",

### Property
| 名称 |类型|备注|
|---|---|---|---|
| id | number |  id |
| name | string |  游戏名 |
| intro | string |  介绍 |
| publisher | string |  发行商 |
| release_time | string | 发售时间|
