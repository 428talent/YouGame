## Summary
修改指定的游戏信息

---

## Request
Path:`/api/game/:id [put] [patch]`

Patch:部分更新

Put:完全更新
### Param

| 名称 |类型|必须|备注|
|---|---|---|---|
| :id | path | 否 | 游戏Id |

### Auth
`JWT token` in Header

### body
#### Example
```json
{
	"name":"new game1",
	"price":5.99,
	"release_time":"2018-10-21",
	"publisher":"R1",
	"intro":"some intro"
	
}
```

#### Property
| 名称 |类型|必须|备注|
|---|---|---|---|
| name | string |Option|  商品名 |
| price | number |Option| 价格 |
| release_time |string| Option | 发行时间 |
| publisher |string| Option | 发行商 |
|intro|string|Option|简介|
