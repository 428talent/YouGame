## Summary
添加至愿望单

**添加至Token所指向的用户**

---

## Request
### Path
`/api/wishlist [post]`

#### Auth
`JWT token` in Header

### body
#### Example
```json
{
	"game_id":2
}
```

#### Property
| 名称 |类型|必须|备注|
|---|---|---|---|
| game_id | number |Require|  游戏ID |

### example

{
