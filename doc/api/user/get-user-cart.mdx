## Summary
获取用户购物车商品信息
## Request

### Path
`/api/user/:userId/carts [get]`

### Parameters

| 名称 |类型|必须|备注|
|---|---|---|---|
| :userId | path | 是 | 要获取的用户Id |

## Request

### Example
```json
{
    "count": 1,
    "page": 1,
    "page_size": 10,
    "next_page": null,
    "prev_page": null,
    "result": [
        {
            "id": 7,
            "good_id": 1,
            "user_id": 3,
            "created": "2018-10-26T06:31:58+08:00",
            "link": [
                {
                    "href": "http://127.0.0.1:8888/api/good/1",
                    "rel": "good",
                    "type": "GET"
                },
                {
                    "href": "http://127.0.0.1:8888/api/user/3",
                    "rel": "user",
                    "type": "GET"
                }
            ]
        }
    ]
}
```

### Property

| 名称 |类型|备注|
|---|---|---|---|
| id | number |  id |
| good_id | number |  商品ID |
| user_id | number |  用户ID |
| created | string |  创建时间 |
