## Summary
获取用户信息  

### Path
`/api/user/:userId [get]`

## Request

### Parameters

| 名称 |类型|必须|备注|
|---|---|---|---|
| :userId | path | 是 | 要获取的用户Id |

## Response

### Example
```json
{
    "id": 3,
    "username": "takayamaaren@gmail.com",
    "last_login": "2018-12-24T05:57:49+08:00",
    "create_at": "2018-10-28T15:41:56+08:00",
    "link": [
        {
            "href": "http://127.0.0.1:8888/api/user/1/profile",
            "rel": "profile",
            "type": "GET"
        }
    ]
}
```

### Property
| 名称 |类型|备注|
|---|---|---|---|
| id | number |  id |
| username | string |  用户名 |
| last_login | string |  最后一次登录时间 |
| create_at | string | 创建时间 |
