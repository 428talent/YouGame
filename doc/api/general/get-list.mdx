## 获取列表
当获取多数据项时使用列表数据进行包装

#### 示例
```json
 "count": 2,
    "page": 1,
    "page_size": 1,
    "next_page": "http://127.0.0.1:8888/api/orders?user=3&state=Created&state=Done&page=2&pageSize=1",
    "prev_page": null,
    "result": [
        {
            ...
        }
    ]
```

#### Property
| 名称 |类型|备注|
|---|---|---|---|
| count | number |  数量 |
| page | number |  页码 |
| page_size | number |  一页的数量 |
| next_page | number |  下一页的url |
| prev_page | number |  上一页的url |
| result | array |结果 |
